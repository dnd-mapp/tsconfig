# @dnd-mapp/tsconfig

Shared, strict `tsconfig.json` presets for Node.js, NestJS, and Angular projects, distributed as a private npm-format package on GitHub Package Registry.

## Language

**GitHub Package Registry**:  
GitHub's own package registry (`npm.pkg.github.com`), used here instead of the public npm registry so the package can stay private to the `dnd-mapp` org. Authenticates with a `read:packages`/`write:packages`-scoped token rather than a npm account.  
_Avoid_: npm registry, the registry (ambiguous with npmjs.com)
